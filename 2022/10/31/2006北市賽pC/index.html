pC.井字遊戲

2022-10-31

95年 北市賽 pC

TIOJ 1061

寫了快100行 😖

Description

輸入9個符號，代表目前棋盤上的情況
假設O為先手，請輸出結束時的以下四項:

  • 所有 不同 的棋面
  • O贏的棋面
  • X贏的棋面
  • 平手的棋面

Solution

由於數字很小，直接用回溯法窮舉即可
每次舉完邊後判斷是否結束(一條線或填滿)，並增加棋面數量
然而很容易有重複的情況發生，因此做一個struct來儲存棋面組合
直接遍歷判斷是否重複即可

如果是空棋盤 總共有958個棋面

其中 O贏626場 X贏316場 平手則有16場

某逃馬技能-先手必勝 🤣🤣🤣

Code

$\color{lime}{AC}$ ( 0.3s , 7mb )
