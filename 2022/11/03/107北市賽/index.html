pC.表演節目 (Show)

TIOJ 2181

超簡單題😌

Discription

資訊能力競賽頒獎典禮將安排一定數量以上的社團進行表演。小朱是這次活動的總策劃，由於臺北市有許多優秀的表演社團，他蒐集過去社團表演的數據，初步篩選出具有高人氣指標的社團，以及該表演的演出費用。請計算在不超過表演總預算的前提下，可邀請表演的社團組合中，社團人氣指標總和最高為多少，並計算其表演所需總費用。
輸入兩數a, N,分別代表「表演總預算 a 千元」$(5 ≤ a ≤ 1000)$,
以及「可被邀請進行表演之社團數量 N」$(1 ≤ N ≤ 1000)$。第$2$至第$N+1$行,分別
列出社團編號$1$至$N$相關資訊,每一行有兩個以空白為間隔的正整數,分別為該社團表演
的人氣指標 $x (1 ≤ x ≤ 100)$ 及演出費用 $y$ 千元 $(1 ≤ y ≤ 100)$。

Solution

基本上就是標準的$01knapsack$問題
只是再額外紀錄花費
$dp$轉移式 $dp[j]=max(dp[j],dp[j-y]+x)$，其中$j$代表該預算能獲得的最大表演酬勞，$x$代表人氣指標，$y$代表演出費用
這裡使用滾動$dp$來記錄數值
記得可能有多個最大人氣指標總和，這裡需要求出最小的成本
這裡題目沒寫，害我直接吃了一個$WA$🙄

Code

$\color{lime}{AC}$ ( 4ms , 3mb )

pD.幸運表格(Lucky)

TIOJ 2182

easy~~~

Discription

給定一個$m(5 < m \le 1000) \times n(5 < n \le 1000)$ 的表格，每一個格子都被預先填入一個整數
可以任意選擇一個點當作起點，每次只能往右或往下移動，直到超出表格為止

Solution

$dp$ 轉移式:

令$dp(i,j)$為以該點為終點的最大路徑
$x=max(dp[i-1][j],dp[i][j-1])$(左邊與上面的格子)

由於可以任意選擇起始表格，因此如果左方和右方皆小於0則可直接拋棄，並選擇該點為起點
最後再尋找在邊界上的最大值 (很重要，路徑需經過邊界)

Code

$\color{lime}{AC}$ ( 60ms , 7mb )
