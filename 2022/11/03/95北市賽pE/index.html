pE.最大矩形(Area)

2022-11-03

95年 北市賽 pE

TIOJ 1063

這題有夠難 😵

Discription

輸入一個 $m \times n$的二維矩陣區域，$0$代表該處有障礙物，$1$代表為空地
假設每個子區域面積皆為$1$，請求出最大的矩陣面積，且該區域內無障礙物

Solution

由於不能選取$0$的區域，列一般的$dp$轉移式有一定難度
不如換個角度思考:

既然不能選$0$，那不如直接把$0$的區域設為很小的負數，再計算二維$LCS$就好了?

如此一來就簡單多了 😋

Code

$\color{lime}{AC}$ ( 12ms , 3mb )
