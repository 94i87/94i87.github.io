pA.飛天李晴(?)(sunny)

2022-11-02

105年 北市賽 pA

TIOJ 1419

有難度的題目😎

Description

飛天桑妮是一隻能在樹木間移動的鼯鼠(?)
她居住的森林有$N$棵樹($\le 10^5$)，
每棵樹$X_i$,$Y_i$($\le 10^4$)和$H_i$($\le 10^5$)為座標及該樹高度
桑妮的家位於$(0,0)$，而桑妮的奶奶家位於的位置$(10000,10000)$
每次從較高的樹跳到較低的樹，即可使出滑翔絕技(?)，享受樂趣
樂趣值為該次旅行中的所有樂趣的最大值\\\\
每次的樂趣為較高的樹減較低的樹的數值，最低為 $0$
需求出旅行最大的樂趣值
(需經過起點和終點)，且每次飛行的樹會距離起點越來越遠，不能回頭

Solution

由於是計算最大值，其實只要經過合法範圍最高的樹和最低的即可
基本上就是要求出兩數 $a,b$，滿足 $\sqrt{X_a^2+Y_a^2} \le \sqrt{X_b^2+Y_b^2}$，使得高度差$H_b-H_a$ 為最大
使用掃描線演算法，由近到遠依序計算
複雜度 $O(nlogn)$
