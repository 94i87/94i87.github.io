104-pC.框架區間

2022-11-01

104年 全國賽 pC

TIOJ 1913

這題其實超水XD
😂😂

Description

輸入T，共有T組測資($T \le 20$)
輸入一個包含N ($1 \le N \le 5000$)個數字且不重複的基因序列 $S$ = ( $s_1$ , $s_2$ , $\cdots$ , $s_n$ )
在基因序列$S$上若有兩個整數$a$和$b$，滿足 $1 \le a < b \leq N$ ，且所有在(兩數字在集合$S$ )之間的數字恰好為$a$~$b$
，稱為「框架區間」 (framed interval)
請求出集合$S$內有多少框架區間 ?

Solution

窮舉所有組合，總共$n^2$次(最大$25000000$)
每次需判斷兩數間的數字是否為$a$~$b$，因此用兩種方式判斷:

  • 該區間總和
  • 該區間最小/最大值

區間總和用前綴和計算，可以方便判斷數字是否正確
但可能會有不在$a$~$b$的數字，所以再額外計算一個最小/大值
因為總和相等，只要有一個數字大於該區間，則必有一數字小於，選一紀錄即可
由於不修改且詢問次數高，使用tarjan的rmq算法來實作

Code

$\color{lime}{AC}$ ( 1.2s , 3mb )
