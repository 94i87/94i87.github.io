pA.售票系統(Sales)

TIOJ 1059

純計算題😗

Description

低利航空公司常給予預購票大幅折扣。該公司所飛行的航班的成本為票面價之30%。
企畫部門提出了以下的調整機位售價的模式：
給定某航段的票面價$s$ ($s \le 5000$)及機位數$t$ ($t \le 250$)，以及$n$ ($n \le 10$)個整數代表每次訂位的機位數，前 20% 賣出的機位售價為票面價之 70% ，前 21%~40% 賣出的機位售價為票面價之 80% ，前 41%~60% 賣出的機位售價為票面價之 90% ，最後售出的機位則全部為票面價。請計算每天每班飛航班機的利潤或虧損

Solution

純計算不太需要什麼技巧
記得題目看清楚，全程用整數運算

Code

pB.函數計算(Comp)

TIOJ 1060

這題根本簽到題

Description

給出一個整數 $x$ ，請求出 $f(x)$
$$
f(x)=
\begin{cases}
\ f(x-1)-h(x) &if \ x \ > h(x) \\
f(g(x))-g(x) &if \ x \ < h(x) \\
1 &otherwise
\end{cases}
$$

$$
h(y)=
\begin{cases}
-1 &if \ y < 2 \\
2+h(y-1)-h(y-2) &otherwise
\end{cases}
$$

$$
g(z)=
\begin{cases}
z^2-1 &if \ z \le 2\\
2 &otherwise\
\end{cases}
$$

Solution

由於可能有負數，因此使用map來儲存
一樣使用top-down $dp$的概念

Code

$\color{lime}{AC}$ ( 0.1s , 17mb )

pC.井字遊戲(TTT)

TIOJ 1061

寫了快100行 😖

Description

輸入9個符號，代表目前棋盤上的情況
假設O為先手，請輸出結束時的以下四項:

  • 所有 不同 的棋面
  • O贏的棋面
  • X贏的棋面
  • 平手的棋面

Solution

由於數字很小，直接用回溯法窮舉即可
每次舉完邊後判斷是否結束(一條線或填滿)，並增加棋面數量
然而很容易有重複的情況發生，因此做一個struct來儲存棋面組合
直接遍歷判斷是否重複即可

如果是空棋盤 總共有958個棋面

其中 O贏626場 X贏316場 平手則有16場

某逃馬技能-先手必勝XD

Code

$\color{lime}{AC}$ ( 0.3s , 7mb )

pE.最大矩形(Area)

TIOJ 1063

這題蠻難的 😵

Discription

輸入一個 $m \times n$的二維矩陣區域，$0$代表該處有障礙物，$1$代表為空地
假設每個子區域面積皆為$1$，請求出最大的矩陣面積，且該區域內無障礙物

Solution

由於不能選取$0$的區域，列一般的$dp$轉移式有一定難度
不如換個角度思考:

既然不能選$0$，那不如直接把$0$的區域設為很小的負數，再計算二維$LCS$就好了?

如此一來就簡單多了 😋

Code
