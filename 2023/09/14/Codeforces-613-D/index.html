Codeforces 613 D

2023-09-14

613-D

這題超難 而且很難找題解 雖然我沒看XD

Description

給定一個 $N$ 個節點的樹，並操作 $Q$ 次詢問
每次詢問給定一數字 $K$，並給予 $K$ 個特殊點
輸出能夠刪除最少的非特殊點數 $S$ 使所有的特殊點皆互不連通

Solution

先考慮一個詢問時的解法
首先若有任何特殊點相鄰，則為無解，輸出 $-1$
將樹 $bottom up dfs$
對於任一點 $V_i$
若 $V_i$ 特殊點，則將該點的子樹的頭(深度最低，連接 $V_i$ 的點)移除若且唯若子樹中至少有一點與 $V_i$ 連通
若 $V_i$ 不為特殊點則將該點移除若且唯若其子樹與 $V_i$ 連通點數至少為 $2$
但若按此方法實行複雜度為 $O(NQ)$
仔細觀察(通靈?)可以發現與所有特殊點有關(會改變解答的點)為所有點和兩兩個點的 $LCA$
設 $V_i$ 為特殊點時，可直接移除子樹的頭(因為所有特殊點不相連)，因此可忽略其他非特殊點
設 $V_i$ 為非特殊點時，因其他非特殊點不影響子樹可連通點大小，因此亦可忽略
先將原樹按 $dfs$ 序排序 $(tin)$
對於任兩個 $dfs$ 序相鄰的點，插入其 $LCA$ 至點集中
至於此作法正確性證明
每兩兩相鄰的點，其 $LCA$ 有兩種

  • 1.其一為另一點祖先，此時 $LCA$ 為特殊點(已在點集中)
  • 2.$LCA$ 為一 $dfs$ 序小於該兩點的一點
    而所有其一點的子孫和另一點的 $LCA$ 皆為該兩點的 $LCA$ ，反之亦然
    而由於 $dfs$ 序為由小到大，因此其他子樹的頭($dfs$ 序與該點不相鄰)與該點的 $LCA$ 皆已計算過(不論深度是否相同)

因此總點數為 $2k-1$
此技巧又稱為輔助樹 ($Auxiliary$ $tree$)
建樹時用單調 $stack$ 維護當前祖先，非祖先直接 $pop$ 掉，並推入該點
以下為複雜度
查找 $LCA$ 及排序為 $O(klogN)$，$dfs$ 為 $O(k)$
總複雜度為 $O(\sum klogN)$;
本題還是很機掰的多筆測資，小心 $initial$ 時吃大大 $\color{red}{TLE}$

Code

$\color{lime}{AC}$ ( 140ms , 38mb )
