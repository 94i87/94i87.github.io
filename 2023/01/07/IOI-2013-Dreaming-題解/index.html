IOI 2013 Dreaming 題解

2023-01-07

TIOJ 1838

第二次解IOI題 花了超久=="

Description

水蛇(Serpent)住在一處擁有$N$個水洞的地方，水洞編號為 $0, …, N - 1$ 。除此之外，有$M$條雙向初始路徑，每條路徑連結一組的水洞讓水蛇可以通過。每一個水洞(不論直接或是間接)至多被一條路徑序列給連結著，但仍然有水洞可能都沒被任何的路徑給連結到 ( 所以 $M ≤ N-1$ )。每條路徑會花掉水蛇特定的天數來通過，通過時間每條路徑有可能會不一樣。
水蛇的好友，袋鼠(Kangaroo)，希望可以挖出 $N - M - 1$ 條新的路徑，讓水蛇可以在任意兩個水洞之間遊走。袋鼠可以在任意兩個水洞之間挖出新的路徑，而由袋鼠挖出的路徑，水蛇需要 $L$ 天就可以通過。
另外，袋鼠希望水蛇在各水洞之間盡可能的快速移動。袋鼠將會挖出新的路徑，使得旅行於任兩個水洞之間所需要的時間最短。在袋鼠挖出新的路徑後，請你幫助水蛇和袋鼠計算出各水洞之間最長移動時間為何。

Solution

由於題目並未規定新的路徑需要連接哪個 $Node$，因此可以假設所有的點皆連在同一個點上，且距離(邊數)皆為$1$ ，而這個點就是樹的圓心(使得所有葉節點的最大距離為最小)，且可以證明圓心必定在最大的樹中
求出圓心後即可直接利用樹 $DP$ 計算
對於所有的節點 $X_i$ ，$dp ( X_i ) $ 表示以節點 $X_i$ 為根的直徑長度， $ dep(X_i) $ 表示以節點 $X_i$為根與其所有葉節點的距離的最大值，對於任何節點 $X_i$，
$dep(X_i)=max(dep(X_j)+dis(X_i,X_j)\ | \ for \ j \ in \ adj[X_i]) $， $dis(X_i,X_j)$ 為兩節點的距離， $adj[X_i]$ 為其所有子節點， 祖先需特判
而直徑會有以下三種可能:

  • 直徑位於原本的圖中，對於所有的節點 $X_i$，直徑等於 $max(X_i)$
  • 直徑位於原本兩個最大的分開的樹中，而圓心在其中一邊，其數值等於 $rad(0)+rad(1)+l$，$l$ 為路的長度，$rad$為所有樹的直徑裡的最大兩個數值
  • 直徑位於原本第二大和第三大的分開的樹中，而圓心並不在任何一個樹中，其數值為 $rad(1)+rad(2)+l*2$
