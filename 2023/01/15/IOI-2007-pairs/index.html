IOI-2007-Pairs 題解

2023-01-15

TIOJ 1345

這次想得比較快 大概只花3小時XD

Description

馬可 (Mirko) 和史拉可 (Slavko) 在玩動物玩具。首先，他們先從如下圖的三種遊戲盤面中選擇一種盤面類型。
盤面上佈滿了一維、二維、或三維的細格（在圖中以圓圈代表之）。

以下圖省略

接著馬可將$N$個動物玩具擺放到細格上。
兩個細格之間的距離定義為動物玩具從其中一格移動到另一細格的最少步驟。
在每一步驟，一個動物玩具可移到與其相鄰（在上圖中以線段連接）的細格。
有趣的是，這些玩具會發出叫聲，（仙女龍?!）(啊ˊˊ啊ˋˋ)
若兩個動物玩具的距離不超過$D$時，他們就可互相聽到對方的叫聲。
史拉可的目標如下：計算有多少組$(pair)$的動物玩具可以聽到互相的叫聲。
請寫一個程式，當給定一種盤面、動物玩具的位置、及數字$D$時，計算出符合上述條件的組數。

Solution

對於所有三種盤面，分別用不同的解法

一維

基本上只要讀入陣列，排序後二分搜即可

二維

這裡假設前面的座標$(a,b)$，當前座標$(x,y)$，最大曼哈頓距離$D$
則可列出所求 $|x-a|+|y-b|\leq D$
假設 $a\le x$
移項可得 $x \pm y \leq a \pm b+d$
只要判斷加減號是否皆正確即可
將$x-y$排序，並使用 $BIT$ 來實作，記得陣列要開兩倍

三維

這個超級麻煩 🤣
一樣假設前面座標 $(a,b,c)$，當前座標 $(x,y,z)$
所求為 $|x-a|+|y-b|+|z-c| \leq D$
一樣設 $a\le x$
移項 $x \pm y \pm z \leq a \pm b \pm c+d$
而這次總共有 $4$ 種組合，每個皆須判斷
因此開一個三維的 $BIT$ 來計算
這邊先將所有值加上 $75$，否則會有小於$0$的情況
將$x-y-z$排序，剩下的丟入 $BIT$ 中即可
而至於如何計算(集合互斥)
總共有 $2^3=8$ 種組合，$x+y+z$ 和 $x-y-z$ 一定是加回去的
剩下的就是 $C_2^8=28$種組合啦 多試一下就過了XD

Code

$\color{lime}{AC}$ ( 2.2s , 52mb )
